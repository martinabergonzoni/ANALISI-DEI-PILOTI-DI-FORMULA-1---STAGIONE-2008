# ANALISI-DEI-PILOTI-DI-FORMULA-1---STAGIONE-2008
Il progetto analizza le performance di tutti i professionisti che hanno gareggiato nel campionato di F1 nel 2008. 

## DESCRIZIONE DEL PROGETTO
Questa analisi rappresenta il primo progetto che ho mai sviluppato in Python. E' stato sviluppato nell'ambito del Master in Data Analytics di ProfAI.
L'obiettivo dell'analisi è offrire insight di base sulle gare di Formula 1 durante l'anno 2008: in particolare, le aree di interesse che ho sviluppato sono:

- performance dei piloti (classifica, punteggi, vittorie, podi)
- performance dei costruttori (punteggio e classifica)

Una sezione divertente è stata implementata all'inizio, dove è possibile inserire il nome del pilota preferito e visualizzare i suoi insight di stagione!

Il progetto possiede, inoltre, una funzione che restuisce all'utente un file txt contenente la classifica d'annata.

## COME USARE IL PROGETTO

1. Nella prima cella ti verrà chiesto di inserire un file per far funzionare il codice. Si tratta del file CSV formula1_data che inserirò tra le risorse.
   formula1_data contiene tutte le informazioni di piloti e costruttori
2. In seguito, ti verrò richiesto di inserire il nome di un pilota di Formula 1 di quell'anno. Una volta inserito, il codice restiruirà informazioni su punti fatti, vittorie e podi raggiunti
3. Al termine del progetto, potrai visionare direttamente tra i download del tuo pc il file Classifica_drivers_2008, contentenente la classifica integrale della stagione Formula 1 2008.
